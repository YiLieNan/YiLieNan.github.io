## 什么是 Hexo？

[Hexo](https://hexo.io/) 是一个快速、简洁且高效的静态博客框架。它基于 Node.js，支持 Markdown 写作，可以一键部署到 GitHub Pages、Vercel 等平台。

## 特性

- ⚡ **极速生成** — Node.js 驱动，生成速度飞快
- 📝 **Markdown 支持** — 用最熟悉的 Markdown 语法写作
- 🎨 **丰富的主题** — 大量精美主题可供选择
- 🔧 **插件生态** — 丰富的插件支持扩展功能
- 🚀 **一键部署** — 支持 Git、FTP、Vercel 等多种部署方式
